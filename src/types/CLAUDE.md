# Types Module

TypeScript 类型定义。

## Files

| File | Description |
|------|-------------|
| `index.ts` | 主导出 |
| `gateway.ts` | 网关相关类型 |
| `service.ts` | 服务相关类型 |
| `mcp.ts` | MCP 协议类型 |

## Key Types

- `GatewayConfig` - 网关配置
- `McpServiceConfig` - MCP 服务配置
- `ServiceInstance` - 服务实例
- `McpMessage` - MCP 消息
- `Logger` - 日志接口
